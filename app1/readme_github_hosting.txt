
Setting up github hosting

https://github.com/sanskrit-lexicon-scans/boesp1/settings
click 'Pages' (at the left)
  https://github.com/sanskrit-lexicon-scans/boesp1/settings/pages
click 'None  (Select branch) -- choose 'main' branch
click 'Save'
 "Your GitHub Pages site is currently being built..."
 This takes a few minutes.
 When done, 
  https://github.com/sanskrit-lexicon-scans/boesp1/settings/pages
  shows
  Your site is live at https://sanskrit-lexicon-scans.github.io/boesp1/

  clicking on this link shows the readme.MD file

Our app uses the url of form (for verse 125):
  https://sanskrit-lexicon-scans.github.io/boesp1/app1/?125
